# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2024-05-22
### Added
- **Architecture**: Completely migrated to Feature-first (Clean Architecture).
- **Database**: Integrated Drift for local persistence and bookmarks.
- **Networking**: Switched from old API to new GNews API using Dio.
- **Localization**: Added support for 3 languages (English, Russian, Turkish).
- **Theme**: Dark and Light mode support with system auto-switching.
- **Notifications**: Local notifications for breaking news via Settings.
- **Security**: Added `.env` support for API keys.
- **UX**: Added Native Splash Screen and custom Flutter animations.

### Changed
- Rebranded from "News App" to "Daily Leaf".
- Refactored state management to use multiple Providers.
- Updated UI with a modern, minimalist design.

### Removed
- Legacy mock data service.

## [1.0.0] - 2023-06-25
- Documentation Update.

## [1.0.0] - 2022-12-03
- Initial release.
